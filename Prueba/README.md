# flutter_application_4

Proyecto Flutter con arquitectura MVC y diseño Atomic Design para verificar si dos números enteros positivos son números amigos.

## Funcionalidad

- Recibe dos números enteros positivos.
- Calcula la suma de los divisores propios de cada número.
- Indica si forman un par de números amigos.

## Ejemplo

- Entrada: 220 y 284
- Salida: son números amigos
